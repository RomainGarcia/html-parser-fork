# HTML Parser

This is a simple HTML parser implemented in PHP. It can be used to display the structure of an HTML document in a readable format.

> [!CAUTION]
> This application is intended to be used for training purposes, and contains intended vulnerabilities. Do not deploy it in a public environment.

## Usage

To use the HTML parser, clone the repo, then, run the following command in the terminal:

```bash
git clone https://github.com/Secureaks/html-parser.git
cd html-parser
composer install
php -S 0.0.0.0 8000 -t public/
```

Then, open your web browser and navigate to `http://localhost:8000`. You will see a form where you can input an HTML document. After submitting the form, the structure of the HTML document will be displayed in a readable format.
